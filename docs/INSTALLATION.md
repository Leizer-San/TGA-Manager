# Установка и первый запуск TGA Manager

## Системные требования

- Windows 10 или Windows 11 x64;
- Python 3.10 или новее;
- подключение к интернету для установки зависимостей и загрузки Telegram Desktop;
- 7-Zip в `PATH` только при импорте `.7z` и `.rar`.

## Автоматическая установка

1. Скачайте исходный код через **Code → Download ZIP**.
2. Распакуйте ZIP в папку, путь к которой не планируется менять.
3. Запустите `install.bat`.
4. Разрешите установку Python через `winget`, если установщик предложит её.
5. Дождитесь сообщения `TGA Manager is ready`.
6. Запустите ярлык **TGA Manager** на рабочем столе.

Не перемещайте папку проекта после создания ярлыка. Если папку всё же перенесли, повторно запустите `install.bat`.

### Установка без ярлыка

```powershell
powershell -ExecutionPolicy Bypass -File .\install.ps1 -NoShortcut
```

После этого запускайте `run.bat`.

## Ручная установка

```powershell
py -3 -m venv .venv
.venv\Scripts\python.exe -m pip install --upgrade pip setuptools wheel
.venv\Scripts\python.exe -m pip install -r requirements.txt
.venv\Scripts\python.exe main.py
```

## Установка Telegram Desktop

1. Откройте **Настройки**.
2. В карточке **Telegram Desktop** нажмите **Скачать**.
3. Выберите автоматический вариант установки.
4. Дождитесь окончания загрузки и распаковки.

Для ручной установки скачайте `tportable-x64.*.zip` на странице [официальных релизов](https://github.com/telegramdesktop/tdesktop/releases), нажмите **Выбрать ZIP** и укажите архив.

После установки должен появиться файл:

```text
Telegram/Telegram.exe
```

## Добавление первого аккаунта

### Существующая TDATA

1. Нажмите **Добавить**.
2. Укажите папку `tdata` или папку, внутри которой она находится.
3. При необходимости задайте SOCKS5-прокси.
4. Нажмите **Авторизация TDATA**.

### Новый вход

1. Нажмите **Добавить → Ручной вход**.
2. Выполните вход в открывшемся Telegram.
3. Дождитесь загрузки списка чатов.
4. Нажмите **Завершить** в TGA Manager.

## Обновление приложения

1. Сохраните папку `data` в резервную копию.
2. Замените исходные файлы приложения новыми, не затрагивая `data` и `Telegram`.
3. Повторно запустите `install.bat` для обновления зависимостей.

## Удаление приложения

TGA Manager не регистрирует системные службы. Для удаления:

1. Закройте все экземпляры Telegram, запущенные менеджером.
2. При необходимости сохраните папку `data`.
3. Удалите ярлык с рабочего стола.
4. Удалите папку приложения.

## Решение проблем

### Повреждено `.venv`

Повторно запустите `install.bat`. Установщик проверит окружение и пересоздаст его, если Python внутри не запускается.

### Python не найден

Установите Python 3.12 с [python.org](https://www.python.org/downloads/windows/) и включите пункт **Add Python to PATH**, затем снова запустите `install.bat`.

### Telegram.exe не найден

Используйте кнопку **Скачать** в настройках либо вручную поместите portable-клиент в `Telegram/Telegram.exe`.

### Не импортируются RAR или 7Z

Установите [7-Zip](https://www.7-zip.org/) и убедитесь, что команда `7z` доступна в PowerShell.
